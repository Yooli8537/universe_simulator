# Info

This File contains information to the different categories in the bodies.py file to help with better understanding.

## Sizes

Sizes are the radius of the particles / circles. The size is stored in a variable which contains a range of float numbers.

## Colors

Colors are stored in a RGB range.

## Density

Each body type has a set multiplier based on their real density. This multiplier is applied to their size to determine their mass.

## Rarity

Rarity is a weighted stat. The higher the weight, the more likely the body will be picked.
