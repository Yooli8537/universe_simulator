# Info

This File contains information to the different categories in the bodies.py file to help with better understanding.

## Sizes

Sizes are the radius of the particles / circles.

- 1 = Tiny
- 3 = Small
- 5 = Medium
- 7 = Large
- 9 = Giant

Numbers between (2, 4, 6, 8) are just extra values to have slightly more variation.

## Colors

Colors are stored in a RGB range.

## Mass / Density

Each body type has a set multiplier based on their real density. This multiplier is applied to their size.
